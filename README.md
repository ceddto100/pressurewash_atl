# Pressure Wash of Atlanta Website

Production website for Pressure Wash of Atlanta, a locally owned residential
pressure washing company serving Metro Atlanta.

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## About

The site highlights residential pressure washing, soft washing, pricing ranges,
service areas, and a quote form that delivers submissions to
cartercedric35@gmail.com.
